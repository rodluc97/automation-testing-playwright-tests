# Playwright Automation Suite

Este proyecto es un framework de automatización robusto utilizando **Playwright** y **TypeScript**. Incluye pruebas de flujos de E-commerce, procesamiento de archivos con IA y manejo de interacciones complejas.

## Características del Proyecto
- **Arquitectura POM (Page Object Model):** Separación clara entre la lógica del test y los elementos de la interfaz.
- **Data-Driven Testing:** Uso de archivos JSON para ejecutar múltiples casos de prueba dinámicamente.
- **Manejo de Archivos:** Automatización de subida de PDFs y validación de resultados generados por IA.
- **Reportes Avanzados:** Generación automática de evidencias (screenshots y videos) en caso de fallos.

## Tecnologías utilizadas
- [Playwright](https://playwright.dev/) - Framework de automatización.
- [TypeScript](https://www.typescriptlang.org/) - Lenguaje de programación.
- [Node.js](https://nodejs.org/) - Entorno de ejecución.
